# Chrome Logger

A Chrome extension that shows console logs and network requests from the browser in a sidebar panel, without needing to to open DevTools.

- Captures `console.log`, `console.warn`, `console.error`, and `console.info` messages
- Monitors and displays api requests and other network activity
- Ability to Toggle visibility without losing captured data
- Ability to turn the logger on/off from the extension seperate popoup window
